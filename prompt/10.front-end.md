프론트(web)에 기본 페이지를 구현하라.

파일:
- web/src/main.tsx
- web/src/routes.tsx
- web/src/api/http.ts (ky 인스턴스: Authorization 헤더 자동 부착)
- web/src/store/auth.ts (zustand로 accessToken 저장)
- web/src/pages/Login.tsx
- web/src/pages/Queue.tsx
- web/src/pages/AdminEventForm.tsx
- web/src/pages/Purchase.tsx
- web/src/pages/Result.tsx

요구사항(요점):
- Login: /auth/login 호출 후 토큰 저장, /queue로 이동.
- Queue: /queue/enqueue → /queue/status 폴링(지수백오프), READY면 gateToken 보관 후 /purchase 이동.
- AdminEventForm: 이벤트 생성 폼(간단 검증).
- Purchase: 수량 입력 → /orders 호출(헤더: Idempotency-Key, X-Gate-Token).
- Result: orderId로 상태 조회 표시.
- React Query로 API 호출, 로딩/에러 상태 처리.
- 전체 코드 출력.
