NestJS(api)에 Auth 모듈을 구현하라.

파일:
- api/src/auth/auth.module.ts
- api/src/auth/auth.controller.ts
- api/src/auth/auth.service.ts
- api/src/auth/jwt.strategy.ts
- api/src/auth/jwt-refresh.strategy.ts (리프레시 토큰)
- api/src/auth/user.entity.ts (이미 있으면 재출력)
- api/src/auth/dto/*.ts (RegisterDto, LoginDto, TokenDto)

요구사항:
- POST /auth/register {email,password} → 200 또는 409.
- POST /auth/login {email,password} → {accessToken, refreshToken}.
- POST /auth/token {refreshToken} → {accessToken}.
- bcrypt로 비밀번호 해시/검증.
- JWT 설정: ACCESS(JWT_ACCESS_SECRET, exp: JWT_ACCESS_EXP), REFRESH(JWT_REFRESH_SECRET, exp: JWT_REFRESH_EXP).
- JwtAuthGuard 헬퍼 제공.
- 에러 메시지/상태코드 명확히 처리.
- 전체 코드 출력.
