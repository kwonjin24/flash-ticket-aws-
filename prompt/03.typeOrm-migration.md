NestJS(api)에서 TypeORM 설정과 엔티티/마이그레이션을 추가하라.

요구사항:
- 파일:
  api/src/config/typeorm.config.ts
  api/src/auth/user.entity.ts
  api/src/events/event.entity.ts
  api/src/orders/order.entity.ts
  api/src/app.module.ts (연결)
- 엔티티 스펙:
  User(id uuid pk, email unique, passwordHash, role USER|ADMIN)
  Event(id uuid, name, startsAt, endsAt, totalQty, soldQty default 0, maxPerUser, price, status DRAFT|ONSALE|CLOSED)
  Order(id uuid, userId, eventId, qty, status HOLD|PAID|CANCELLED|EXPIRED, amount, idemKey nullable unique where not null, createdAt now())
- 마이그레이션 SQL 파일(경로: api/migrations/xxxx_init.sql)도 함께 출력.
- app.module.ts에서 ConfigModule + TypeOrmModule.forRootAsync 사용. synchronize=false, migrationsRun=true.
- 전체 코드 출력.
