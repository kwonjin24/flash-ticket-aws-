요구사항:
- POST /payments {orderId, method:'MOCK'} → {paymentId, status:'REQ'}
- POST /payments/callback (내부) {paymentId, orderId, status:'OK'|'FAIL'}
- status=OK일 때:
  - 트랜잭션으로 orders.status=PAID, events.soldQty += qty (행 잠금).
  - 멱등: 콜백 중복 호출 시 재커밋 금지.
- 전체 코드 출력.
