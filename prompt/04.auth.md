Gateway 서버에 Auth 모듈을 구현하라.

파일:
- gateway/src/presentation/auth/auth.module.ts
- gateway/src/presentation/auth/controllers/auth.controller.ts
- libs/auth/src/**/* (DTO, Service, Facade, Guard, Strategy)
- api/src/presentation/auth/auth.module.ts (Access 토큰 검증 전용 모듈)

요구사항:
- Gateway `/auth/register`, `/auth/register/admin`, `/auth/login`, `/auth/refresh`, `/auth/logout` 제공.
- `libs/auth` 공용 계층에서 DTO/서비스/가드/전략을 관리하고 Gateway/Auth 모듈에서 재사용.
- JWT 설정: ACCESS(JWT_ACCESS_SECRET, exp: JWT_ACCESS_EXP), REFRESH(JWT_REFRESH_SECRET, exp: JWT_REFRESH_EXP).
- bcrypt로 비밀번호 해시/검증.
- API 서버는 Access 토큰 검증만 수행하도록 AuthModule 구성.
- 에러 메시지/상태코드 명확히 처리.
- 전체 코드 출력.
