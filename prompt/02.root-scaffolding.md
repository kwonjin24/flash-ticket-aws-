단일 레포 구조를 만든다. 다음 폴더/파일을 생성하고 초기화하라.

- 루트(package.json, docker-compose.yml, .env.example, README.md)
- api/ (NestJS 프로젝트 골격)
- web/ (Vite React TS 프로젝트 골격)

요구사항:
- 루트 package.json에 스크립트:
  "dev": "concurrently -k \"pnpm --dir api start:dev\" \"pnpm --dir web dev\"",
  "db:up": "docker compose up -d",
  "db:down": "docker compose down -v"
- docker-compose.yml: Postgres 16, Redis 7.
- .env.example: PORT=4000, JWT 키들, DB/REDIS 설정, CLIENT_URL.
- api: @nestjs/config, @nestjs/typeorm, pg, bcrypt, @nestjs/jwt, passport-jwt, ioredis, bullmq 설치를 전제로 config/모듈 파일 생성.
- web: react-router-dom, @tanstack/react-query, ky 설치를 전제로 기본 라우팅(Login/Queue/AdminEventForm/Purchase/Result) 스켈레톤 생성.
- 각 파일 전체 코드를 출력.
