파일:
- api/src/events/events.module.ts
- api/src/events/events.controller.ts
- api/src/events/events.service.ts
- api/src/events/dto/*.ts (CreateEventDto, UpdateEventDto)

요구사항:
- POST /events (ADMIN만) {name, starts_at, ends_at, total_qty, price, max_per_user}
- GET  /events/:id (공개)
- PATCH /events/:id (ADMIN만) 일부 수정
- 상태 전이 보조 메서드: DRAFT→ONSALE→CLOSED(간단 검증)
- 전체 코드 출력.
